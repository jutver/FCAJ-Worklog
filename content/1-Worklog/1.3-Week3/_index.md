### Week 3 Achievements:

* Grasped the fundamentals of cloud computing and the broad architecture of the AWS ecosystem.
* Recognized and distinguished between key AWS service pillars like Compute, Storage, Networking, and Databases.
* Set up and tuned an AWS Free Tier account successfully.
* Navigated the AWS Management Console with ease, locating services and controlling primary resources.
* Successfully deployed and set up the AWS CLI, adjusting:
  * Access Keys
  * Secret Access Keys
  * Default Regions
  * Output formats
* Executed core CLI commands to:
  * Verify account details and configurations
  * Display accessible AWS regions
  * Inspect EC2 instances
  * Generate and handle key pairs
  * Track active services and resources
* Acquired practical EC2 skills through:
  * Deploying instances
  * Establishing SSH connections
  * Provisioning and attaching EBS volumes
* Comprehended core networking components:
  * VPC layout and subnetting
  * Contrasting Security Groups with Network ACLs
  * The functions of Internet Gateways and NAT Gateways
* Successfully implemented a foundational multi-tier network design.
* Cultivated the capability to administer AWS environments using both the Management Console and the CLI concurrently.
